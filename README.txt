INSTALLATION INSTRUCTIONS
==========================================

1. UPLOAD TO SERVER:
   - Upload the entire "modular-theme" folder to:
     /wp-content/themes/modular-theme/
   
2. ACTIVATE IN WORDPRESS:
   - Go to WordPress Admin > Appearance > Themes
   - Find "Modular" theme
   - Click "Activate"

3. VERIFY:
   - Check that blocks appear in Gutenberg editor (MODULAR category)
   - Verify pages load correctly
   - Check for any PHP errors in error logs

==========================================
TROUBLESHOOTING
==========================================

If blocks don't appear:
1. Clear browser cache (Ctrl+Shift+R / Cmd+Shift+R)
2. Clear WordPress cache
3. Verify ACF Pro is active
4. Check PHP error logs

If theme doesn't activate:
1. Check file permissions (755 for folders, 644 for files)
2. Verify all files uploaded correctly
3. Check PHP version compatibility

To revert:
- Simply switch back to previous theme in Appearance > Themes
